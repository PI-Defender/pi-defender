OB_PREOP_CALLBACK_STATUS K_OB::_PreObCallback ( _In_ PVOID  pRegistrationContext,
_In_ POB_PRE_OPERATION_INFORMATION  pPreOperationInfo 
)
staticprivate

We get notify when a process want a handle.


We communicate with the user-mode service to know if this process is signed.
If the process is signed, we don't filter.
Else, we filter it and remove 'PROCESS_VM_WRITE' and 'PROCESS_VM_OPERATION' to the desired access.

Parameters
[in]pRegistrationContext- Context that the driver want to pass to this callback.
[in]pPreOperationInfo- Pointer to OB_PRE_OPERATION_INFORMATION structure, that receive information about the handle created.
Returns
OB_PREOP_SUCCESS - Pre Operation callback finished.
