BOOLEAN K_HELPER::IsValidStringArray ( _In_ const _KEY_VALUE_PARTIAL_INFORMATION *  pInfo)
static

Returns TRUE if the passed _KEY_VALUE_PARTIAL_INFORMATION structure extracted from the Registry contains a valid Unicode string array (REG_MULTI_SZ type), else returns FALSE
The REG_MULTI_SZ type is a sequence of null-terminated strings, terminated by a empty string (\0).

ex: String1\0String2\0String3\0LastString\0\0

Parameters
[in]pInfo- pointer to a _KEY_VALUE_PARTIAL_INFORMATION that contains data extracted from a value in the Registry
Returns
TRUE - The provided structure contains a valid Unicode string array (REG_MULTI_SZ). FALSE - The provided structure does not contain a valid Unicode string array (REG_MULTI_SZ).
